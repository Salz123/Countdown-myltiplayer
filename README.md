# Countdown App (React + Vite)

## Local run
1) Install Node.js 18+
2) In this folder:
   - npm install
   - npm run dev

## Build for static hosting (recommended for Hostinger)
- npm run build
This creates /dist. Upload the *contents* of /dist to Hostinger hPanel -> File Manager -> public_html.

## If you use Hostinger Node.js App
You can upload the whole project and run:
- npm install
- npm run build
- npm run preview (or configure to serve /dist with a static server)
